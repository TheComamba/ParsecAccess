# ParsecAccess

A Rust crate to access the [PARSEC database](https://people.sissa.it/~sbressan/CAF09_V1.2S_M36_LT/) for stellar evolutionary trajectories.

## Usage

In your `Cargo.toml`, include the `parsec_access` crate:

```toml
[dependencies]
parsec_access = "0.1.0"
```
